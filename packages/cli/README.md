# @specferret/cli

SpecFerret CLI for spec drift detection.

## Install

```bash
bun install -g @specferret/cli
```

## Usage

```bash
ferret init
ferret scan
ferret lint
ferret review
```

## Runtime

The CLI is built for Bun.
